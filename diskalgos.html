Operating System - Disk Scheduling Algorithms

FCFS

Track Requests are taken in the order they arrived in the Disk Queue.

SSTF

Track Requests which are closer to the current request are served first.

SCAN

Track Requests from one End to other End are serviced in between.

CSCAN

As soon as Head reaches the other End, it immedaitely returns to the beginning of the track.

LOOK

Head starts from the first request at one end of the disk and moves towards the last request at the other end servicing all the requests in between.

CLOOK

Head starts from the first request at one end of the disk and moves towards the last request at the other end servicing all the requests in between.
